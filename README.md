# Name My Pet (v1.1.20)

AI-Powered Pet Naming Application.

## Deployment Details
- **Framework**: React 18.2.0
- **Build Engine**: Vite 5.1.6 (HTML5 Standard)
- **Runtime**: Browser-native ESM

## Local Setup
1. `npm install`
2. `npm run dev`
