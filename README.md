# Name My Pet

AI-Powered Pet Naming Application.

## Deployment Environment (v1.1.19)
- **Framework**: React 18.2.0 (Strict Mode)
- **Build Engine**: Vite 5.1.6
- **Runtime**: ESM-only via Browser Import Maps

## Local Development
1. `npm install`
2. Set `GEMINI_API_KEY` in `.env`
3. `npm run dev`

---
*Note: This repository is configured for automatic deployment to Vercel.*
